Typography

Keeping in line with UCF's brand guidelines, Athena's base font family is Gotham. For users that cannot subscribe to Cloud.typography's webfont services, a fallback font specifically for Athena can be used instead. Please see the installation/getting started page for more information.

Note that Athena does not provide webfonts out of the box; it is your responsibility to install them on your site using either a Cloud.typography CSS key, or the fallback webfont <link> tag provided on the installation page.
