A Note About Bootstrap:

Athena is a complete replacement for Bootstrap. If you are already using Bootstrap on an existing project and want to use Athena instead, you should remove Bootstrap's CSS and Javascript completely. Athena is designed to run as a completely standalone framework.

Quick Start

To quickly add all of Athena's features to your project, add the following to your pages:

  1. Add webfonts: Add a <link> tag into the document <head> before all other stylesheets to apply custom webfonts to the page. If you're using Cloud.typography, this is the CSS Key that is provided whenever you create a new project. See the webfont configuration section for information on setting up a Cloud.typography project for use with Athena.

    To use the fallback sans-serif font library, use the following <link> tag: 

    <link rel="stylesheet" href="TODO/webfont-fallback.css">

  2. Add framework stylesheet: Copy the <link> tag below and paste it into the document <head> immediately after the Cloud.typography/fallback font <link> tag: 

    <link rel="stylesheet" href="TODO/framework.min.css">

  3. Add JavaScript: Add our JavaScript plugins, jQuery, and Tether near the end of your pages, right before the closing </body> tag. Be sure to place jQuery and Tether first, as our code depends on them. While we use jQuery's slim build in our docs, the full version is also supported. 

    <script src="https://code.jquery.com/jquery-3.1.1.slim.min.js" integrity="sha384-A7FZj7v+d/sdmMqp/nOQwliLvUsJfDHW+k9Omg/a/EheAdgtzNs3hpfag6Ed950n" crossorigin="anonymous"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/tether/1.4.0/js/tether.min.js" integrity="sha384-DztdAPBWPRXSA/3eYEEUWrWCy7G5KFbe8fFjk5JAIxUYHKkDx6Qin1DkWx51bBrb" crossorigin="anonymous"></script>

Webfont Configuration

The preferred sans-serif font family for Athena is Gotham. Gotham can be installed in your project via the Cloud.typography service.

Gotham Licensing for the Web

Any existing print license for Gotham usage, whether purchased via the UCF Computer Store or separately, does not cover usage of webfonts. To use Gotham on the web, you must register for the Cloud.typography service. Do not attempt to self-host Gotham font files online; doing so will breach the university's font license agreement.

For base support with Athena, we recommend configuring your Cloud.typography project to include the following fonts/weights:

  • Gotham ScreenSmart Light
  • Gotham ScreenSmart Light Italic
  • Gotham ScreenSmart Book
  • Gotham ScreenSmart Book Italic
  • Gotham ScreenSmart Bold
  • Gotham ScreenSmart Bold Italic
  • Gotham ScreenSmart Black
  • Gotham ScreenSmart Black Italic

Because Athena does not support older browsers, enabling Legacy Support is not necessary and is not recommended.

More weights can be added as needed per-project, however, we don't recommend usage of Gotham ScreenSmart Extra Light due to readability concerns.

Avoid adding webfonts that aren't needed for your project; as the number of fonts in a project increases, the average load time for the font project increases.

Using Fallback Webfonts

UCF Marketing provides a fallback sans-serif webfont, based on the free font family Montserrat, intended specifically for use with Athena. This font should only be used for web projects when registering to use Gotham on the web is not possible. Do not use this font for any other purpose besides as a web font.

The fallback sans-serif webfont comes prepackaged with light, book, semi-bold, bold, extra-bold, and black font weights.

See the Quick Start section for fallback webfont installation instructions.
