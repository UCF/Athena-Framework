Navbars

NOTE: Navbars have been modified in Athena with updates that are not currently available in the Bootstrap alpha-6. Notably, .navbar-toggler is not positioned via -left/right classes in Athena; the toggler button is aligned by markup order. Athena currently supports right-aligned toggler buttons: if they are added after a .navbar-brand (or other supported nested components), it will be positioned correctly without any additional markup; if the toggler button is the only child besides the collapsed content, you will need to add the .ml-auto class to push it to the right. Left-aligned toggler buttons are not currently supported in Athena.
