Athena Project

The Athena Project is a CSS and Javascript framework based heavily on Bootstrap themed and customized for use on University of Central Florida sites.

The following pages, act as a set of test pages for viewing the default styles provided by Athena. They are broadly organized on the Bootstrap 4 documentation to make finding components easy.

Want to contribute?

If you're interested in contributing the Athena Project, please feel free to visit our GitHub Page and send us a message.
