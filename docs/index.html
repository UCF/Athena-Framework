Athena Framework

UCF's Web Brand Toolkit

Overview

The Athena Framework is a front-end framework built and used by the UCF Marketing Team to create responsive websites and web applications for UCF. Based on Bootstrap 4, the framework includes most of the functionality developers and content editors have come to expect of Bootstrap. In addition, base Bootstrap styles have been updated to conform to the UCF brand by default, with additional variants for flexibility.
